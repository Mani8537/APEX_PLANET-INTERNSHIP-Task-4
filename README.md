# 📊 Data Storytelling & Statistical Validation

## 📌 Project Overview

This project focuses on transforming customer data into meaningful
business insights using data storytelling and statistical validation
techniques.

## 🎯 Objective

-   Identify factors influencing revenue performance\
-   Understand customer churn behavior\
-   Validate insights using statistical methods

## 📂 Dataset Description

Includes Age, Gender, Revenue, Churn, Payment Method, Customer Segment
and more.

## 📊 Business Problem

-   What drives revenue?\
-   What influences churn?

## 📈 Statistical Analysis

### Test 1: Age vs Revenue (T-Test)

-   Variables: Age, Revenue\
-   Result: p-value \< 0.05 (Significant)

### Test 2: Gender vs Churn (Chi-Square)

-   Variables: Gender, Churn\
-   Result: p-value \> 0.05 (Not Significant)

## 🔍 Key Insights

-   Age impacts revenue\
-   Gender does not affect churn

## 📌 Business Conclusions

-   Focus on age-based segmentation\
-   Use behavioral data for churn reduction

## 🚀 Recommendations

-   Target high-value age groups\
-   Improve marketing strategies

## ▶️ Deliverables

-   PPT Presentation\
-   README\
-   LinkedIn Video

## 📌 Conclusion

Data storytelling + statistical validation enables better business
decisions.
